Kebijakan Privasi

Terakhir diperbarui: 27 Juli 2025

1. Pendahuluan

HijraPlan adalah aplikasi perencanaan keuangan pribadi dan bisnis yang Islami. Kami menghargai privasi Anda dan merancang aplikasi ini tanpa sistem akun atau pendaftaran. Anda dapat langsung menggunakan aplikasi dan memilih mode yang sesuai: Pribadi atau Bisnis.

2. Pengumpulan Data

HijraPlan tidak mengumpulkan data pribadi. Semua data keuangan yang Anda masukkan disimpan hanya di perangkat Anda.

  • Kami tidak meminta nama, email, atau informasi identitas lainnya.
  • Data Anda tidak dikirimkan ke server manapun.
  • Aplikasi tidak memerlukan koneksi internet untuk digunakan.

3. Mode Penggunaan

Pengguna dapat memilih antara mode Personal dan Bisnis saat pertama kali menggunakan aplikasi. Pilihan ini dapat diubah kapan saja melalui pengaturan.

4. Penyimpanan Data

Semua data keuangan Anda disimpan secara lokal di perangkat Anda. Kami tidak memiliki akses terhadap data tersebut. Jika Anda menghapus aplikasi atau membersihkan data, maka semua catatan keuangan Anda akan ikut terhapus.

5. Keamanan Data

Karena data disimpan secara lokal, keamanan bergantung pada sistem operasi perangkat Anda. Kami menganjurkan untuk mengaktifkan fitur keamanan perangkat seperti PIN, sidik jari, atau enkripsi.

6. Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Perubahan penting akan diinformasikan melalui pembaruan aplikasi.

7. Kontak

Jika Anda memiliki pertanyaan atau masukan, silakan hubungi kami melalui email: support@hijraplan.id.
